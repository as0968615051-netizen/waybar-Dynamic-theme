# waybar-Dynamic-theme
Waybar Dynamic Theme: A Kid's take on Absolute Flat Dynamic Island.
🛠️ How to Use

1. Move style.css and config.jsonc to ~/.config/waybar/.
2. Restart Waybar to apply the changes.
